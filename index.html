Ski2000

GitHub Live

Ski2000 is a browser based skiing game based on classic 1980's
ski games and built using JavaScript and Phaser.

The goal of the game is simple. Try to accumulate 2000 points by passing through gates and avoiding fatal encounters with trees and snowmonsters.

RubyChess

GitHub

RubyChess is a console based chess game written in (surprise!) Ruby. The game features four major classes: piece, board, display and game. RubyChess uses parent/child relationships to alert players of valid moves as well as check and checkmate.
